aquaponics-node-red
===================

The node-red component of the aquaponics control system


dependencies
============

* node-js
* node-red (v0.4.0)
* pause node
* wiring-pi 
* aquaponics-arduino

installation notes
==================
npm install node-red
npm install wiring-pi
